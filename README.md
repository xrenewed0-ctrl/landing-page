## 📋 Customization Guide

### Change Business Information
Open `index.html` and find these sections:
- Navigation, hero section, about, contact
- Replace phone number, address, hours as needed
- Update Google Maps link in the contact iframe

### Change Colors
Open `style.css` and modify CSS variables:
```css
:root {
    --primary-color: #00d4ff;      /* Change glow color */
    --secondary-color: #0099cc;    /* Change accent */
    --dark-bg: #0a0e27;            /* Change background */
    --text-light: #e0e0e0;         /* Change text */
}
```

### Add/Remove Sections
Edit `index.html`:
- Add new sections for blog, testimonials, etc.
- Update navigation menu links

### Update Chatbot Responses
Open `script.js` and edit `chatbotResponses` object:
```javascript
const chatbotResponses = {
    'your-keyword': 'Your response text',
    // Add more keywords...
};
```

### Add Images
Replace placeholder SVG images with real images:
```html
<!-- Old -->
<img src="data:image/svg+xml,..." alt="Description">

<!-- New -->
<img src="path/to/image.jpg" alt="Description" class="lazy-image">
```

## 🔐 Security Notes

- Contact forms submit to console (no real backend)
- In production, implement:
  - Server-side form validation
  - CSRF protection
  - Rate limiting
  - Email verification

### To Enable Real Form Submission:
1. Set up a backend server (Node.js, PHP, Python, etc.)
2. Update `script.js` contact form handler:
```javascript
contactForm?.addEventListener('submit', async (e) => {
    e.preventDefault();
    const response = await fetch('/api/contact', {
        method: 'POST',
        body: new FormData(contactForm)
    });
    // Handle response
});
```
